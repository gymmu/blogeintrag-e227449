Einleitung

Geräteturnen ist eine durchaus bekannte Sportart. Sie ist eine einfachere Variante von Kunstturnen, um Kindern und jugendlichen eine ähnliche Disziplin zu bieten ohne dass sie proffessionell trainieren müssen. Auch im Schulsport ist das Geräteturnen eine beliebte Note um Koordination, Beweglichkeit und Kraft zu prüfen.

Aufbau

Im Geräteturnen gibt es verschidene Kategorien, angefangen in dem K1. In jeder Kategorie gibt es verschiedene Schwierigkeitsstufen. Das bedeutet das man je höher die Kategorie ist mehr und schwierigere Elemente lernen.
